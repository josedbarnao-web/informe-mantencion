Acumular en Google Sheets — configuración (una sola vez)
  1. En console.cloud.google.com, crea o elige un proyecto, y activa la Google Sheets API (menú "APIs y servicios" → "Habilitar APIs").
  2. En "APIs y servicios" → "Credenciales" → "Crear credenciales" → ID de cliente de OAuth, tipo "Aplicación web". En "Orígenes de JavaScript autorizados" agrega la URL exacta donde esta app va a estar alojada (ej. https://tuusuario.github.io) — no funciona abriendo el archivo directo con doble clic (file://), tiene que estar publicada en una URL real.
  3. Copia el Client ID que te genera y pégalo abajo.
  4. Crea una Hoja de Cálculo de Google nueva (o usa una existente) para que sea la base acumulada. Ábrela, y copia el ID que aparece en la URL entre /d/ y /edit (ej. docs.google.com/spreadsheets/d/ESTE_ID/edit). Pégalo abajo también.
  5. Comparte esa hoja (botón "Compartir") con todas las cuentas de Google que vayan a usar esta app, con permiso de Editor.
